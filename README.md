# Blinkit Data Analysis
## Project Objective
To create an end-to-end data analysis project using Excel, SQL, Python, and Power BI.
The goal is to clean the data, analyze it, and build a clear dashboard that helps understand important insights.

## Process
- Identified key metrics like sales trends, item performance, and outlet insights.
- Cleaned and prepared the Blinkit dataset using Excel, SQL, and Pandas.
- Analysed category-wise, rating-wise, and outlet-type patterns.
- Built a clear dashboard to highlight major sales trends and business insights.
